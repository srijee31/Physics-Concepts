# Scalars and Vectors

## Introduction

In physics, quantities are classified into two types based on the information needed to describe them.

1. Scalar Quantities
2. Vector Quantities

The key difference is whether **direction** is required or not.

---

## Scalar Quantities

A **scalar quantity** is a physical quantity that has **only magnitude** and **no direction**.

It can be completely described using just a number and a unit.

### Examples

- Mass → 5 kg
- Time → 10 s
- Temperature → 30 °C
- Distance → 100 m
- Speed → 60 km/h

Example explanation:

If we say **"The temperature is 25°C"**, we only need the value and unit.  
There is **no direction involved**.

---

## Vector Quantities

A **vector quantity** is a physical quantity that has **both magnitude and direction**.

To completely describe a vector, we must specify:

- Magnitude
- Direction

### Examples

- Displacement
- Velocity
- Acceleration
- Force
- Momentum

Example explanation:

If we say **"A force of 10 N towards east"**, we have:

Magnitude → 10 N  
Direction → East

Both are required to fully describe the quantity.

---

## Representation of Vectors

Vectors are usually represented using **arrows**.

The arrow shows:

- Length → magnitude
- Arrow direction → direction of the vector

Vectors can also be written in notation form:

Example:

Vector A → **A**

Sometimes written as:

**A⃗**

---

## Difference Between Scalar and Vector Quantities

| Property | Scalar | Vector |
|--------|--------|--------|
| Magnitude | Yes | Yes |
| Direction | No | Yes |
| Representation | Number + unit | Arrow or vector notation |

Examples:

Scalar → mass, time, temperature, speed  
Vector → displacement, velocity, force

---

## Key Points

- Scalar quantities have **only magnitude**.
- Vector quantities have **magnitude and direction**.
- Vectors are represented using **arrows**.
- Understanding vectors is important for studying **motion, force, and mechanics**.
