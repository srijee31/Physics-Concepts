## 📌 Speed vs Velocity

### 🚗 What is Speed?

Speed is the **rate of change of distance**.

- It tells **how fast** an object is moving
- It is a **scalar quantity** (no direction)
- Always **positive**

### 🧮 Formula:
Speed = Distance / Time

👉 Example:  
If a car travels 100 meters in 10 seconds,  
Speed = 100 / 10 = **10 m/s**

---

### 🎯 What is Velocity?

Velocity is the **rate of change of displacement**.

- It tells **how fast AND in which direction**
- It is a **vector quantity** (has direction)
- Can be **positive, negative, or zero**

### 🧮 Formula:
Velocity = Displacement / Time

👉 Example:  
If a car moves 100m east in 10 seconds,  
Velocity = **10 m/s east**

---

## 🔍 Key Differences

| Feature        | Speed                      | Velocity                     |
|---------------|---------------------------|------------------------------|
| Type          | Scalar                    | Vector                       |
| Direction     | ❌ No                     | ✅ Yes                       |
| Based on      | Distance                  | Displacement                 |
| Sign          | Always positive           | +, -, or 0                   |
| Formula       | Distance / Time           | Displacement / Time          |

---

## 🧠 Simple Analogy

Imagine you're going to college:

- Speed = how fast you are going 🚀  
- Velocity = how fast + **which direction** (towards college 🎯)

---

## ⚡ Important Points

- Speed ≥ Velocity (in magnitude)
- If direction is constant:
  👉 Speed = Velocity
- If object changes direction:
  👉 Speed ≠ Velocity

---

## 🎯 Real-Life Example

A runner completes one full lap in a circular track:

- Speed = total distance / time  
- Velocity = 0 (because displacement = 0)

---

## 🔥 Types of Speed

1. **Uniform Speed** → Constant speed  
2. **Non-uniform Speed** → Changing speed  
3. **Average Speed** = Total Distance / Total Time  
4. **Instantaneous Speed** → Speed at a particular moment  

---

## 🚀 Summary

- **Speed** = how fast  
- **Velocity** = how fast + direction
