# Accuracy vs Precision

## What is it?

Accuracy and precision are terms used to describe the quality of measurements.

---

## Simple Meaning

👉 Accuracy = how close a value is to the true value  
👉 Precision = how close repeated values are to each other  

---

## Example

- True value = 10 cm  

Case 1:
9.9, 10.1, 10.0 → Accurate and Precise ✅  

Case 2:
8.5, 8.6, 8.5 → Precise but not Accurate ❌  

Case 3:
9.0, 10.5, 11.0 → Neither accurate nor precise ❌  

---

## Why it is used?

- To evaluate measurement quality  
- To improve experimental results  
- To reduce errors  

---

## Key Points

- Accuracy = closeness to true value  
- Precision = consistency of values  
- High precision does not mean high accuracy  
- Both are important in experiments
