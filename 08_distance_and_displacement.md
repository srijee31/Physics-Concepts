## 📌 Distance vs Displacement

### 🚶‍♂️ What is Distance?

Distance is the **total path** travelled by an object.

- It is a **scalar quantity** (only magnitude, no direction)
- Always **positive**
- Depends on the actual path taken

👉 Example:
If you walk 5m forward and 3m backward,  
Distance = 5 + 3 = **8 meters**

---

### 🎯 What is Displacement?

Displacement is the **shortest straight-line distance** from starting point to ending point.

- It is a **vector quantity** (has magnitude + direction)
- Can be **positive, negative, or zero**
- Does NOT depend on path, only start & end points

👉 Example:
If you walk 5m forward and 3m backward,  
Displacement = 5 - 3 = **2 meters (forward direction)**

---

## 🔍 Key Differences

| Feature        | Distance                     | Displacement                  |
|---------------|----------------------------|-------------------------------|
| Type          | Scalar                     | Vector                        |
| Direction     | ❌ No                      | ✅ Yes                        |
| Path Dependent| ✅ Yes                     | ❌ No                         |
| Value         | Always positive            | Can be +, -, or 0            |
| Formula       | Total path length          | Final position - Initial position |

---

## 🧠 Simple Analogy

Imagine going from **home → shop → home**

- Distance = total walking (to shop + back home)
- Displacement = 0 (because you ended where you started)

---

## ⚡ Important Points

- Distance ≥ Displacement always
- Displacement can be zero, even if distance is not
- If motion is in straight line & one direction:
  👉 Distance = Displacement

---

## 🎯 Real-Life Example

A person runs around a circular track and comes back to start:

- Distance = circumference of track
- Displacement = 0

---

## 🚀 Summary

- **Distance** = how much ground covered  
- **Displacement** = how far from starting point (with direction)
