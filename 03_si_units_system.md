# SI Units System

## What is it?

SI Units System (International System of Units) is a standard system used worldwide to measure physical quantities.

---

## Simple Meaning

👉 A common set of units used everywhere to measure things.

---

## Example

- Length → meter (m)  
- Mass → kilogram (kg)  
- Time → second (s)  

---

## Why it is used?

- To maintain uniform measurements worldwide  
- To avoid confusion between different unit systems  
- Used in science, engineering, and daily life  

---

## Key Points

- SI stands for International System of Units  
- It has standard units for all measurements  
- Used globally  
- Based on 7 fundamental units
